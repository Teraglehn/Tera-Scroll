TeraScroll

Un système de scrollbar développé par Teraglehn, simple, rapide à utliser, configurable et surtout ergonomique, ce script répond aux attentes des plus fainéant comme des plus rigoureux.
Compatibilité :

Démo

Prérequis, Jquery, Jquery UI, etc...

Afin de faire fonctionner le script de scrollbar vous aurez besoin d'inclure dans votre page certaine bibliothèques, bien connu pour certaine, et bien moins pour d'autre.

Jquery

Vous aurez besoin de la bibliothèque Jquery 1.7+, pour l'inclure dans votre page, deux solution :

  • Téléchargez la bibliothèque ici et chargez la dans votre page depuis votre site :

    <script type="text/javascript" src="chemin/vers/jquery.js"></script>

  • Liez la bibliothèque depuis l'API google :

    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7/jquery.min.js"></script>

L'avantage de l'API google est que beaucoup de site lient Jquery depuis ce lien, ce qui vas donc favoriser la mise en cache de la bibliothèque dans le navigateur du client.

Jquery UI

Vous aurez également besoin de la bibliothèque Jquery UI, mais uniquement du module draggable

  • Téléchargez la bibliothèque ici et chargez la dans votre page depuis votre site :

    <script type="text/javascript" src="chemin/vers/jqueryui.js"></script>

    Vous pouvez ne choisir que le module draggable (avec ses dépendances), ou bien prendre toute la bibliothèque.
  • Liez la bibliothèque depuis l'API google :

    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jqueryui/1.8/jquery-ui.min.js"></script>

L'avantage de l'API google vaut également pour la bibliothèqe Jquery UI.


MouseWheel 3.0.6, by Brandon Aaron

Ce plugin permet de gérer la molette de la souris, si vous ne souhaitez pas utiliser l'utiliser dans vos scrollbar, inutile de télécharger ce plugin.
Donc pour inclure ce script dans votre page :

<script type="text/javascript" src="chemin/vers/jquery.mousewheel.min.js"></script>

TeraScroll 1.0

Voilà donc le coeur du sytème, il contient toute les fonction nécessaire au fonctionnement des scrollbar, à télécharger ici.
Et pour inclure le script dans votre page :

<script type="text/javascript" src="chemin/vers/scroll.js"></script>

Résumé :

L'inclusion des différent script doit néanmoins être fait dans un certaine ordre, voici celui conseillé :

<!DOCTYPE html>
<html>
	<head> ... </head>
	<body>
		...

		<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7/jquery.min.js"></script>
		<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jqueryui/1.8/jquery-ui.min.js"></script>
		<script type="text/javascript" src="chemin/vers/jquery.mousewheel.min.js"></script>
		<script type="text/javascript" src="chemin/vers/scroll.js"></script>
	 </body>
</html>

Il n'y a que deux règles :

  • Jquery doit être le premier lien.
  • Scroll.js doit être le dernier lien.

Configuration, mise en place

CSS


CSS par defaut


Le pack contient un css minimaliste, contenant le design d'une scrollbar, et le css nécessaire au fonctionnement du système :

Code : CSS
.scrollable {
	/* Nécessaire */
	overflow: hidden;
}
.scrollbar {
	/* Design */
	border-radius: 20px;
	background-color: rgba(0, 0, 0, 0.5);
	cursor: pointer;

	/* Doit être spécifié */
	width:5px;

	/* Nécessaire */
	float:left;	
}

Si la largeur de la scrollbar est "0px", alors elle sera invisible, elle ne pourras être déplacée à l'aide du clic souris, mais la molette souris fonctionnera.

Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.

Javascript

La scrollbar minimum est initialisée à l'aide de la fonction scrollable(), en passant en paramettre un identifiant sous forme de chaine de caractères :

Code : Javascript
$(document).ready( function(){
	scrollable("#Ma_Div");
});

Vous pouvez ajouter un deuxième argument, afin d'avoir plusieurs scrollbar différente, entrez le nom de la classe :

Code : Javascript
$(document).ready( function(){
	scrollable("#Ma_Div", "scrollbar2");
});
La classe par défaut est "scrollbar".
Enfin vous pouvez ajouter un troisième argument, qui permet de changer la classe du conteneur :

Code : Javascript
$(document).ready( function(){
	scrollable("#Ma_Div", false, "scrollable2");
});
La classe par défaut est "scrollable".
